# Nitroglycerin

- **Indication** - Angina, acute coronary syndrome
- **Type** - Nitrate vasodilator
- **Administration**
  - **Sublingual** - 0.3-0.6 mg, repeat every 5 minutes as needed, maximum 3 doses
